void cmu_open ( void  )

Sets up the clocks required for the LETIMER PWM setup.

This function disables the default oscillators in order to select our proper base clock and to minimize power usage. It also enables the higher frequency peripheral clock for the LEDs.

Note
Clocks should always be initialized first, so this function should run prior to the initializing and setup of the LETIMER and other peripherals.
